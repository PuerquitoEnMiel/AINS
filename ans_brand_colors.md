## Typography (Standardized for ANS)
*   **Headings**: Montserrat / Vollkorn (as defined in other ANS projects)
*   **Body**: Inter / System Default

## UI/UX Best Practices for ANS Portals
*   Use **ANS Dark Green (#007934)** for primary headers, navigation bars, and active states.
*   Use **ANS Orange (#FF8300)** for prominent calls to action (CTAs), warnings, or highlights.
*   Use **ANS Light Green (#80BC00)** for success states, secondary buttons, or soft accents.
*   Use ample whitespace, subtle border-radius (`rounded-4` or `rounded-lg`), and soft shadows (`shadow-sm` or `shadow`) to create depth and a modern feel.
